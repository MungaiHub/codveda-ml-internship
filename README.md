## ✅ Tasks Completed

### 🟢 Level 1 — Basic

#### Task 1: Data Preprocessing for Machine Learning
- **Dataset:** Iris Dataset
- **What I did:**
  - Handled missing values (mean/median filling)
  - Encoded categorical variables (Label Encoding & One-Hot Encoding)
  - Normalized and standardized numerical features (MinMaxScaler & StandardScaler)
  - Split dataset into training and testing sets (80/20)
- **Tools:** Python, Pandas, Scikit-learn

#### Task 3: K-Nearest Neighbors (KNN) Classifier
- **Dataset:** Iris Dataset
- **What I did:**
  - Trained a KNN model on the labeled Iris dataset
  - Tested K values from 1 to 20 and identified the best K
  - Evaluated performance using accuracy, confusion matrix, precision, recall, and F1-score
  - Compared results across different K values with visualizations
- **Tools:** Python, Scikit-learn, Pandas, Matplotlib

---

### 🟡 Level 2 — Intermediate

#### Task 2: Decision Trees for Classification
- **Dataset:** Iris Dataset
- **What I did:**
  - Trained a full unpruned decision tree and visualized its structure
  - Pruned the tree using max_depth tuning (tested depth 1–10)
  - Compared full tree vs pruned tree performance
  - Evaluated using accuracy, F1-score, and confusion matrix
  - Performed feature importance analysis
- **Tools:** Python, Scikit-learn, Pandas, Matplotlib

#### Task 3: K-Means Clustering
- **Dataset:** Churn Prediction Data
- **What I did:**
  - Preprocessed and scaled the dataset using StandardScaler
  - Applied K-Means clustering and determined optimal K using the Elbow Method and Silhouette Score
  - Visualized clusters using PCA 2D scatter plots
  - Interpreted cluster profiles using a heatmap
- **Tools:** Python, Scikit-learn, Matplotlib, Seaborn

---

### 🔴 Level 3 — Advanced

#### Task 2: Support Vector Machine (SVM) for Classification
- **Dataset:** Churn Prediction Data
- **What I did:**
  - Trained SVM models with 4 kernels: linear, RBF, poly, sigmoid
  - Compared kernel performance using accuracy, precision, recall, and F1-score
  - Evaluated the best model using AUC and ROC Curve
  - Visualized the decision boundary using PCA 2D reduction
- **Tools:** Python, Scikit-learn, Pandas, Matplotlib

#### Task 3: Neural Networks with TensorFlow/Keras
- **Dataset:** Iris Dataset
- **What I did:**
  - Designed a feed-forward neural network: 4 → 64 → 32 → 16 → 3 neurons
  - Used ReLU activation, BatchNormalization, and Dropout to prevent overfitting
  - Trained the model using Adam optimizer and backpropagation
  - Applied early stopping to avoid overfitting
  - Visualized training/validation loss and accuracy curves
  - Evaluated using accuracy, F1-score, and confusion matrix
- **Tools:** Python, TensorFlow/Keras, Pandas, Matplotlib
